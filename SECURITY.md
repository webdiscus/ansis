# Security Policy

## Supported Versions

| Version  | Supported          |
|----------| ------------------ |
| >= 3.x.x | :white_check_mark: |
| < 3.0.0  | :x:                |

## Reporting a Vulnerability

If you have discovered a security issue, please report it by creating an [issue](https://github.com/webdiscus/ansis/issues).

Thanks for helping.
